Padding

Gets the number of bits of padding present in the payload.

Declaration
public uint Padding { get; }
Property Value
Type Description
System.UInt32
Remarks

The 6-bit ASCII encoding NMEA uses for the AIS payload can only encode data in multiples of 6 bits. If the underlying message is not a multiple of 6 bits long, the encoding will result in the payload having some extra bits on the end. This property reports how many of those bits there are.
